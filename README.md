# Vida em Movimento — Website Institucional

Este projeto consiste no desenvolvimento do site institucional da ONG Vida em Movimento, com foco em apresentação da organização, divulgação de projetos, estímulo ao voluntariado e recebimento de doações.

O site foi desenvolvido com design responsivo, modo claro/escuro, formulário validado e carregamento dinâmico de conteúdo.


# Tecnologias Utilizadas

**HTML5** | Estruturação das páginas 
**CSS3**  | Estilos, layout responsivo e modo claro/escuro 
**JavaScript (Vanilla)** | Interatividade, máscaras, validação e SPA parcial
**Git & GitHub** | Controle de versão 


# Funcionalidades

- Layout totalmente responsivo
- Tema escuro e claro com persistência via localStorage
- Menu mobile abrindo via toggle
- Formulário completo com:
  -- Máscara automática para CPF, telefone e CEP
  -- Validação com mensagens de erro
  -- Modal de sucesso
- SPA parcial para carregamento de projetos sem recarregar a página
- Código CSS, HTML e JS minificados (versões .min.css, .min.js e .min.html)
